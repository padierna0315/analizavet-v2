# flujo-laboratorio-completo change

This directory contains the implementation of the 'flujo-laboratorio-completo' change.

## What was implemented

This change implemented a complete laboratory file processing workflow that allows uploading HL7 batch files via web interface with automatic processing.

## Components implemented

1. HL7 Batch Splitter (`app/satellites/ozelle/batch_splitter.py`)
2. Reception endpoints and templates
3. Image extraction and triage functionality
4. Complete workflow from file upload to PDF generation
